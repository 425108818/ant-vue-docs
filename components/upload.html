API

urImageUpload

属性说明类型可选值默认值
modelValue（v-model）上传成功后的FileCodestring--
name图片标题string--
imageSize图片显示大小number-174
limitUploadSize限制上传文件大小Number-10 * 1024 * 1024 单位字节

urImageUpload

属性说明类型可选值默认值
fileCode（v-model）上传成功后的FileCodestring--
fileName（v-model）上传成功后的文件名称string--
fileList（v-model）多文件上传绑定的参数。多文件上传时只需v-model:fileList即可FileList[]--
btnName自定义上传按钮名称string-Upload
accept上传文件类型string-['.csv', '.xls', '.xlsx', 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet', 'application/vnd.ms-excel'].join(', ')
limitUploadNum限制上传文件数量Number--
limitUploadSize限制上传文件大小Number-10 * 1024 * 1024

更多属性请参考 Upload 上传。
