Form 表单

提示

该组件只通过ur-form-group(antd Grid 栅格)提取公共样式。如title、间距、字体大小等

基本用法

单据信息
元
收货信息
请选择省市区
展开查看
vue
<template>
  <ur-form>
    <ur-form-group title="单据信息">
      <a-col :span="8">
        <a-form-item label="原始单号">
          <a-input v-model:value="formData.originalOrderSerialNumber" placeholder="请输入原始单号" :maxlength="50" />
        </a-form-item>
      </a-col>
      <a-col :span="8">
        <a-form-item label="期望送达时间" required>
          <ur-date-picker v-model="formData.exceptDeliveryTime" show-time />
        </a-form-item>
      </a-col>
      <a-col :span="8">
        <a-form-item label="商品采购价" required>
          <ur-input-int
            v-model:value="formData.value"
            :rate="100"
            suffix="元"
            placeholder="请填写商品采购价"
            :min="0"
            :step="0.1"
            :precision="2"
            nullable
          />
        </a-form-item>
      </a-col>
      <a-col :span="8">
        <a-form-item label="是否提货">
          <a-switch v-model:checked="formData.isNeedTransport">
            <template #checkedChildren><check-outlined /></template>
            <template #unCheckedChildren><close-outlined /></template>
          </a-switch>
        </a-form-item>
      </a-col>
    </ur-form-group>
    <ur-form-group title="收货信息">
      <a-col :span="8">
        <a-form-item label="配送日期">
          <ur-range-picker v-model:fromDate="formData.value2" v-model:toDate="formData.value3" />
        </a-form-item>
      </a-col>
      <a-col :span="8">
        <a-form-item label="期望送达时间" required>
          <ur-date-picker v-model="formData.exceptDeliveryTime" show-time />
        </a-form-item>
      </a-col>
      <a-col :span="8">
        <a-form-item label="收货地址" required>
          <ur-area-cascader v-model="formData.addressCode" :level="5" />
        </a-form-item>
      </a-col>
    </ur-form-group>
  </ur-form>
</template>

<script setup lang="ts">
import { CheckOutlined, CloseOutlined } from '@ant-design/icons-vue';
import { ref } from 'vue';

const formData = ref({
  isNeedTransport: false,
  originalOrderSerialNumber: null,
  exceptDeliveryTime: null,
  value: null,
  value2: null,
  value3: null,
  addressCode: '310114102',
});
</script>

API

UrForm

UrForm组件与antd Form 表单属性一致。 更多属性请参考antd Form 表单。

UrFormGroup

属性说明类型可选值默认值
title组合标题string--
gutter栅格间隔，可以写成像素值或支持响应式的对象写法来设置水平间隔 { xs: 8, sm: 16, md: 24}。或者使用数组形式同时设置 [水平间距, 垂直间距]number-[20, 16]

更多属性请参考antd Grid 栅格。
