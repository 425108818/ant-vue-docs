API

urSelect

属性说明类型可选值默认值
modelValue（v-model）选中的值string | number--
options选择框数据，默认格式{name,value}[]，自动绑定value值。any[]-[]
filterOption是否开启前端输入过滤数据,默认通过name搜索，可以通过传入函数自定义。（运程搜索与此只能用一个,false远程搜素，true和Function前端过滤）。boolean | function-false
url传入url后，自动请求数据。string--
initData是否组件渲染完成后，请求接口获取数据。为true和url都存在时才会请求数据boolean-true
searchKey远程搜索传接口的搜索的字段string--
bodyParams远程搜索额外参数。object-{}
handleOptionFunc对内置请求到的数据进行额外处理函数。function(option)=>{}[]
optionAttribute自定义显示与绑定字段OptionAttribute-{lable:name, value:value}
placeholderLabellable占位符显示。只有名称存在，没有value时。或远程搜索option不存在该条数据，无法通过value回显时，通过lable赋值该字段回显string | number--

urSelectSearch

属性说明类型可选值默认值
modelValue（v-model）选中的值string | number--
filterOption是否开启前端输入过滤数据,默认通过name搜索，可以通过传入函数自定义。（运程搜索与此只能用一个,false远程搜素，true和Function前端过滤）。boolean | function-false
placeholderLabellable占位符显示。只有名称存在，没有value时。或远程搜索option不存在该条数据，无法通过value回显时，通过lable赋值该字段回显string | number--

更多属性请参考antd Select 选择器。
