Input 输入框

urDatePicker、urRangePicker 只添加倍率转化、小数点限制， 更多功能配置如同antd Input 输入框。

基本用法

rate倍率转化、precision保留2位小数点使用案例

API

UrInputInt

属性说明类型可选值默认值
value（v-model）输入框数字number-0
rate比率number-1
precision保留几位小数点number-0
nullable是否为空值boolean-false

urRangeInputInt

属性说明类型可选值默认值
minValue（v-model）输入最小值number-
maxValue（v-model）输入最大值number-

更多属性请参考antd Input 输入框。
