API

UrPageTable

属性说明类型可选值默认值
pageable是否显示分页boolean-true
paginator与后台交互分页参数字段UrApiPaginator-true
paginatorSizes分页选择器展示选择值string[]-['15', '30', '50', '100']
switchLayout是否显示切换上下布局按钮boolean-false
rowClassName表格行的类名Function(record, index):string-已设置默认背景颜色
useCurrentChangeFunc是否使用内置的分页函数处理boolean-true
columnsKey列设置的列展示缓存key（全局不可以重复，避免在缓存中出现重复key缓存冲突问题。）string--
maxColumnWidth自定义整个表格，最大列宽显示。（添加宽度自适应后，列宽有200px最大显示限制。）string--
scrollX自定义scrollXstring | boolean-max-content

更多属性请参考antd Table 表格。

UrTableTabs

属性说明类型可选值默认值
tabs标签显示{name,value}[][]
field搜索的字段,绑定在路由地址上的字段名string-

更多属性请参考antd Tabs 标签页。
